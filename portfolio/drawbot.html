Drawbot

Drawbot is an RC car that follows a path you draw on a canvas in your webbrowser. I built drawbot using a rapsberry pi nano wireless. The raspberry pi is running a node server hosting a web-site over the lan its connected to. You connect your device to the same local network and you can control the robot remotely with a gamepad controller or by drawing on the canvas. The gamepad functionality comes from the use of the browser's gamepad API. The node server uses a web-socket to communicate to the user allowing realtime control of the bot over wifi. The node server also uses a library for accessing the GPIO pins on the board for controlling two DC motors.
